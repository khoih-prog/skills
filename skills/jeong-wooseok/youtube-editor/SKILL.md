## üõ†Ô∏è Prerequisites

Before running this skill, you need a few things:

### 1. System Tools
You must have **FFmpeg** installed on your system (for audio processing).

- **Ubuntu/Debian:**
  ```bash
  sudo apt update && sudo apt install -y ffmpeg
  ```
- **macOS:**
  ```bash
  brew install ffmpeg
  ```
- **Windows:**
  Download from [ffmpeg.org](https://ffmpeg.org/download.html) and add it to your PATH.

### 2. API Keys
Add these keys to your OpenClaw environment (`~/.openclaw/.env`) or export them in your shell.

- **`OPENAI_API_KEY`**: Required for Whisper (transcription) and GPT-4 (analysis).
- **`NANO_BANANA_KEY`**: Required for AI thumbnail generation.

## üöÄ Usage

### Option 1: Process a YouTube Video
Simply provide the URL. The skill will download the best quality available.

```bash
uv run ~/.openclaw/workspace/skills/youtube-editor/scripts/process_video.py --url "https://youtube.com/watch?v=YOUR_VIDEO_ID"
```

### Option 2: Process a Local File
If you already have the video file on your computer.

```bash
uv run ~/.openclaw/workspace/skills/youtube-editor/scripts/process_video.py --input "/path/to/your/video.mp4"
```

## ‚ùì Troubleshooting

**Error: `FFmpeg not found`**
- Make sure you installed FFmpeg and it's available in your system PATH.
- Try running `ffmpeg -version` in your terminal to verify.

**Error: `OPENAI_API_KEY environment variable is not set`**
- Check your `.env` file or export the key manually: `export OPENAI_API_KEY=sk-...`

**Thumbnail generation failed?**
- Ensure you have the `nano-banana-pro` skill installed and your `NANO_BANANA_KEY` is valid.
