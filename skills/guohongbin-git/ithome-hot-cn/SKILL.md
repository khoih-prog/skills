# IT之家热门资讯监控

IT之家科技新闻热门监控，支持数码评测、产品资讯。

## 功能

### 热门资讯
- **今日热榜** - 今日最热文章
- **数码评测** - 产品评测汇总
- **手机新闻** - 手机行业动态

### 分类内容
- **手机** - 手机产品、品牌动态
- **电脑** - PC、笔记本
- **智能硬件** - 智能设备
- **汽车** - 智能汽车

## 使用方式

### 获取热门资讯

```

```
