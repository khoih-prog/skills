# Automation Workflow Skill

## Core Capabilities
- **Data Scraping + Analysis + Reporting**: Extract data from websites, analyze trends, generate automated reports
- **Social Media Content Automation**: Schedule posts, monitor engagement, auto-generate content
- **Customer Support Automation**: Handle FAQs, route tickets, provide 24/7 support

## Usage Examples
"使用Automation工作流技能从[电商网站]抓取[产品价格]数据，分析[价格趋势]并生成[周报]"
"使用Automation工作流技能为[公司Twitter账号]自动发布[每日行业新闻]，监控[品牌提及]并回复[常见问题]"
"使用Automation工作流技能处理[客户支持]请求，自动回答[订单状态查询]和[退货政策]问题"

## Output Formats
- Automated reports (PDF/HTML)
- Scheduled social media posts
- Customer support responses with escalation paths

## Enterprise Features
- Multi-step workflow orchestration
- Error handling and retry logic
- Audit logging and compliance tracking
- Role-based access control
