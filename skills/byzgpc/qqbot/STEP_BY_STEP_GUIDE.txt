# ClawHub 发布 - 图文操作指南

## 第 1 步：点击 Publish
在 ClawHub 网站顶部导航栏，找到并点击 "Publish" 或 "发布 Skill"

预期看到：上传文件界面

---

## 第 2 步：上传文件

1. 点击 "选择文件" 或 "Upload" 按钮
2. 在弹出的文件选择器中，导航到：
   - 位置：`~/Desktop/`
   - 文件：`qqbot-v1.0.0.zip`
3. 选中文件，点击 "打开"

预期看到：文件名显示在界面上，可能有文件大小显示（14K）

---

## 第 3 步：填写基本信息

### 名称 (Name)
在名称输入框中输入：
```
qqbot
```

### 版本 (Version)
在版本输入框中输入：
```
1.0.0
```

### 描述 (Description)
在描述文本框中粘贴以下内容（全部复制）：
```
QQ 官方机器人配置指南，包含完整部署流程和常见问题解决方案

一键配置 QQ 官方机器人，支持私聊、群聊、频道消息。

功能特点:
✅ WebSocket 实时连接
✅ 支持私聊、群聊、频道消息
✅ 内置 AI 处理器接口
✅ 完整的故障排除指南
✅ 自动 IP 白名单监控脚本

使用场景:
- QQ 机器人开发
- OpenClaw 接入 QQ
- 自动化客服
- 社群管理

安装命令: openclaw skill install qqbot
```

---

## 第 4 步：填写元数据

### 作者 (Author)
```
小皮
```

### 许可证 (License)
下拉选择：`MIT`

### 标签 (Tags)
逐个添加以下标签：
1. 输入 `qq` → 点击添加
2. 输入 `bot` → 点击添加
3. 输入 `im` → 点击添加
4. 输入 `机器人` → 点击添加
5. 输入 `qq-bot` → 点击添加

### 分类 (Category)
下拉选择：`IM` 或 `通讯` 或 `Messaging`

---

## 第 5 步：提交

1. 检查所有信息是否填写正确
2. 勾选 "我同意服务条款"（如果有）
3. 点击 "Submit" 或 "发布" 按钮

---

## ✅ 完成

预期看到：
- "发布成功" 或 "提交审核成功" 提示
- 或者跳转到 "我的 Skills" 页面，显示 qqbot 正在审核中

---

## 📝 发布后

审核通常需要几分钟到几小时。审核通过后：
- 其他用户可以通过 `openclaw skill install qqbot` 安装
- 你的 Skill 会出现在 ClawHub 市场中

---

## 🆘 如果遇到问题

**问题 1：文件上传失败**
- 检查文件大小（应小于 50MB）
- 检查文件格式（应为 .zip）

**问题 2：名称已被占用**
- 尝试使用 `qqbot-cn` 或 `qq-official-bot`

**问题 3：描述字数限制**
- 如果太长，简化为：
  ```
  QQ 官方机器人配置指南，支持私聊、群聊、频道消息，包含完整部署文档。
  ```

---

**准备就绪！请按照以上步骤操作 🎉**
